## ✅ Worker Fixtures = per-worker environment

## 🔧 Example

```ts
import { test as base } from '@playwright/test';

export const test = base.extend({
  workerUser: [async ({}, use) => {
    const user = await createUser();
    await use(user);
  }, { scope: 'worker' }]
});
```

---

## 🔥 Usage

```ts
test('test1', async ({ workerUser }) => {
  // same user reused in worker
});
```

---

## 🔥 When to use

|Scope|Use|
|---|---|
|test|strict isolation|
|worker|performance|

## 🔥 Elite Pattern

- worker → login once
- reuse session

```ts
test.use({ storageState: 'auth.json' });
```
