**Grafana = Visualization + Alerting tool**

### 👉 What it does:

- Turns data into **dashboards (graphs, charts)**
    
- Shows trends like:
    
    - test failures 📉
        
    - flake rate ⚠️
        
    - execution time ⏱️
        
- Sends **alerts** when thresholds are crossed
    

### 🧠 In SDET:

> “Grafana helps you _see_ test health”

**Prometheus = Metrics collection + storage system**

### 👉 What it does:

- Collects data like:
    
    - test results
        
    - API latency
        
    - failures
        
- Stores it as **time-series data**
    
- Queries it using **PromQL**
    

### 🧠 In SDET:

> “Prometheus _stores and processes_ test/system metrics”

# 🔗 How They Work Together

```
Tests → Metrics → Prometheus → Grafana → Dashboard + Alerts
```

- Prometheus = **brain (data)**
    
- Grafana = **face (visuals)**
