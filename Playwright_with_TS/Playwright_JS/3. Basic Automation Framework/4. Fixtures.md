Documentation -> https://playwright.dev/docs/test-fixtures

Playwright Test is based on the concept of test fixtures. Test fixtures are used to establish the environment for each test, giving the test everything it needs and nothing else.

There are 2 types of Fixtures
1. Built-In Fixtures
2. Custom Fixtures

### Built In Fixture
Here `page` is a built-in Fixuture
```TypeScript title="test1.spec.ts"
import {test,expect} from '@playwright/test';

test('basic test',async({page})=>{
	await page.goto('https://google.com');	
});
```

### Creating your own fixture 

```Bash
project/
│
├── fixtures/
│   └── my-fixture.ts
│
├── tests/
│   └── login.spec.ts 

```

```Typescript title="my-fixture.ts"
import {test as base} from '@playwright/test';

export const test = base.extend({

	MyFixture: async({page}, use)=>{
	await page.goto('https://saucedemo.com');
	await use(page);

	},
});

```

```Typescript title="test1.spec.ts"
import {test} from '../fixtures/my-fixture';

test('using my own fixture',async ({MyFixture})=>{
	const title =  await MyFixture.title();
	console.log(title);

});
```
