17. How to build a “Salesforce Test Data Factory” using the API so your Playwright tests create complex data (Accounts → Opportunities → Products → Quotes) in seconds instead of minutes.

A **Salesforce Test Data Factory** lets your Playwright tests create **complex business data through the Salesforce REST API** instead of the UI.

Without it, tests often spend **1–3 minutes creating Accounts, Opportunities, Products, Quotes through UI flows**. With a factory, the same setup takes **1–3 seconds**.

The idea is to create a **reusable API layer that builds data hierarchies automatically**.

---

# 1. What a Salesforce Test Data Factory Does

Instead of this UI flow:

```text
Create Account
Create Opportunity
Add Product
Create Quote
Activate Quote
```

Your test calls:

```javascript
const quoteData = await testDataFactory.createQuoteScenario();
```

And the factory automatically creates:

```text
Account
└── Opportunity
      └── OpportunityLineItem (Product)
            └── Quote
```

---

# 2. Basic Salesforce API Client

First create a reusable API wrapper.

`services/salesforceApi.js`

```javascript
export class SalesforceApi {

  constructor(request) {
    this.request = request;
  }

  async create(objectName, data) {
    const response = await this.request.post(
      `/services/data/v60.0/sobjects/${objectName}`,
      { data }
    );

    const result = await response.json();
    return result.id;
  }

}
```

Usage:

```javascript
const accountId = await api.create('Account', { Name: 'Test Account' });
```

---

# 3. Build the Test Data Factory

Now create a **factory layer that builds relationships**.

`factories/testDataFactory.js`

```javascript
export class TestDataFactory {

  constructor(api) {
    this.api = api;
  }

  async createAccount() {
    return await this.api.create('Account', {
      Name: `Account-${Date.now()}`
    });
  }

  async createOpportunity(accountId) {
    return await this.api.create('Opportunity', {
      Name: 'Test Opportunity',
      StageName: 'Prospecting',
      CloseDate: '2026-12-31',
      AccountId: accountId
    });
  }

}
```

---

# 4. Add Product Creation

Products require **Product2 + PricebookEntry**.

Example:

```javascript
async createProduct() {

  const productId = await this.api.create('Product2', {
    Name: 'Test Product',
    IsActive: true
  });

  const pricebookEntryId = await this.api.create('PricebookEntry', {
    Pricebook2Id: 'STANDARD_PRICEBOOK_ID',
    Product2Id: productId,
    UnitPrice: 100,
    IsActive: true
  });

  return pricebookEntryId;
}
```

---

# 5. Add Opportunity Line Item

Link product to opportunity.

```javascript
async addProductToOpportunity(opportunityId, pricebookEntryId) {

  return await this.api.create('OpportunityLineItem', {
    OpportunityId: opportunityId,
    PricebookEntryId: pricebookEntryId,
    Quantity: 1,
    UnitPrice: 100
  });

}
```

---

# 6. Create Quote

Next create the quote.

```javascript
async createQuote(opportunityId) {

  return await this.api.create('Quote', {
    Name: 'Test Quote',
    OpportunityId: opportunityId,
    Status: 'Draft'
  });

}
```

---

# 7. Create Full Data Scenario

Now combine everything.

```javascript
async createQuoteScenario() {

  const accountId = await this.createAccount();

  const opportunityId = await this.createOpportunity(accountId);

  const pricebookEntryId = await this.createProduct();

  await this.addProductToOpportunity(opportunityId, pricebookEntryId);

  const quoteId = await this.createQuote(opportunityId);

  return {
    accountId,
    opportunityId,
    quoteId
  };

}
```

Your test gets **all related records instantly**.

# 8. Use It in Playwright Tests

Example test:

```javascript
test('Approve Quote', async ({ page, request }) => {

  const api = new SalesforceApi(request);
  const factory = new TestDataFactory(api);

  const data = await factory.createQuoteScenario();

  await page.goto(`/lightning/r/Quote/${data.quoteId}/view`);

  await page.getByRole('button', { name: 'Submit for Approval' }).click();

});
```

This removes **minutes of UI setup**.

---

# 9. Add Data Variants

Factories should support scenarios.

Example:

```javascript
async createOpportunityScenario(type) {

  const accountId = await this.createAccount();

  if (type === "enterprise") {
    return await this.createOpportunity(accountId, 100000);
  }

  if (type === "retail") {
    return await this.createOpportunity(accountId, 10000);
  }

}
```

Now tests simply choose scenarios.
