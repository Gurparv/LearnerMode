First commands 
```Bash
#Installs and starts a container
docker run -d -p 8800:80 httpd 

#Tells currently running 'container processes'
docker ps
# Tells container present in memory(running + dormant)
docker ps -a

# If you want to check if the service is running
curl localhost:8800

# To stop the running container 
# container names are the ones which are funny run-time created one.
docker stop <container_name_or_id>
# OR
docker container stop <container_name_or_id>
```
