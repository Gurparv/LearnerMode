
DOM -> "Document Object Model" 
Structured representation of HTML Documents . Allows javascript to access HTML elements and Styles to manipulate them.

DOM is created as soon as HTML files is received by the browser. 
**DOM is stored in a Tree structure in which each HTML element is an object**

```html
<html>
	<head>
		<title> HELLO </title>
	</head>
	<body>
		<section>
			<p>A paragraph </p>
			<p>paragraph 2 </p>
		</section>
	</body>
</html>
```

```Tree title="DOM Tree Structure"
Document
└── html
    ├── head
    │   └── title
    │       └── "HELLO"
    └── body
        └── section
            ├── p
            │   └── "A paragraph"
            └── p
                └── "paragraph 2"

```

You can manipulate DOM using JS because DOM and DOM methods are part of WEB APIs which are like libraries that browsers implements and we can access from our JavaScript code.
Link -> https://developer.mozilla.org/en-US/docs/Web/API/Document_Object_Model

```Javascript title=Basic DOM

//Using DOM methods and properties for DOM manipulation
const myText = document.querySelector('<CSS-Selector>').textContent;
console.log(myText);

document.querySelector('<Css-selector>').textContent = "Heyo👋" ;
document.querySelector('<Css-selector>').textContent = 10 ;

//To access values or name etc of it
const myVal = document.querySelector('<Css-selector>').value;
const myVal2 = document.querySelector('<Css-selector>').name;
const myVal3 = document.querySelector('<Css-selector>').className;

// To check the Type 
const myVal = document.querySelector("#login-button").className;
console.log(typeof myVal); //string

// To convert String to Number
let MyNum = '15';
//console.log(typeof MyNum);
const MyNum2 = Number(MyNum);
console.log(typeof MyNum, typeof MyNum2); //number string
```

####  Adding an Event  in DOM
```Javascript title="Handling Click Events"
// Adding an Event
document.querySelector(".submit-button").addEventListener('click', function(){
    console.log("I was pressed!");
});
//OR 
document.querySelector(".submit-button").addEventListener('click', ()=>{
    console.log(document.querySelector('<CSS Selector>').value);
});
```

####  Handling a Keyboard Event
Keyboard Click is a 'Global' event and it is attached to `whole document` instead of any particular Element.
```Javascript
//Generic Key Press event listener
document.addEventListener('keydown', function(){
   console.log("A key was pressed"); 
});

// ESC Key Event Listener
document.addEventListener('keydown', function(e){ 
console.log(e);
console.log(e.key);
if(e.key === 'Escape'){
    console.log("Esc was pressed");
    }
});
```
