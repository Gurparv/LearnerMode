## Naming Convention in JS is `camelCase`
```Javascript
// variable
let firstName = "John";
const myAge = 20;
// constants
const MAX_LIMIT = 100;
// function
function getUserData(){}
// classes -> PascalCase
class UserProfile{}
// interfaces PascalCase
interface UserData{}
// Types / Enums -> Pascal Case
type UserRole = "admin" | "user";
enum Status {Active, Inactive}

// Private/Internal fields
class A {
private _count = 0;
#realPrivate = 1;
}
// Files and Module Names -> kebab-case or camelCase
user-service.ts //prefer this in modern projects
userService.ts

```
