## Destructuring Arrays
```Javascript
'use strict';

const names = ['tim', 'john', 'monty' ];
console.log(names);

const a = names[0];
const b = names[1];
const c = names[2];

// Destructuring Arrays
const [x, y, z] = names;
console.log(x,y,z);  // tim john monty
const [name1 , name2] = names;
console.log(name1, name2); // tim john

const [name1, , name2] = names;
console.log(name1, name2); // tim monty

const age = [15, 18];

```

##### Use-case -> Inverting 2 variable values
==**Mutating a variable** means altering the value or internal state of an object that the variable references, rather than reassigning the variable to point to a completely new object.


## The Spread Operator (...)
- Used to unpack elements . 
-  It works on all iterables in Javascript eg Arrays, Strings, maps and sets but not objects.
- Note - With new version of ES -> spread operator now supports objects too.


```Javascript
'use strict';

const name = 'Timy';
console.log(name);  // Timy
console.log(...name); // T i m y

const ages = [19,15,10,19,18];
console.log(ages); // [19, 15, 10, 19, 18]
console.log(...ages); // 19 15 10 19 18

const newAges = [10, 20, 30, ...ages];
console.log(newAges); // [ 10, 20, 30, 19, 15, 10, 19, 18]
console.log(newAges.length); // 8

//Note
console.log(`${...name}`); //error 
```

## Rest Pattern and Parameters
Used to pack items, basically opposite of `Spread operator`
```Javascript
'use strict';

// SPREAD , on the right side of '='
const myArray = [1,2,3,9,9,9];
console.log(myArray); // [ 1, 2, 3, 9, 9, 9 ]
console.log(...myArray); 1 2 3 9 9 9

// REST, on the left side of '='
const [a,b,...others] = myArray;
console.log(a,b,others); // 1 2 [ 3, 9, 9, 9 ]
console.log(others); // [ 3, 9, 9, 9 ]

const basicDetail = {
	student1 : {
		name: 'Tim',
		age: 20
	}
};

// collecting items from object using Rest
const {college=false, ...student1} = basicDetail.student1;
console.log(student1);
console.log(college);
```

##### Accepting multiple parameters In function using Rest
```javascript
'use strict';

function sum(...numbers){
	console.log(numbers);
	let total = 0;
	for (let i=0;i<numbers.length;i++){total = total + numbers[i]};
	console.log(total);
}

sum (2,3);
sum (5,5,6);
sum (10,20,30,40,50,60);
const myNumbers = [1,2,0,8,8,1,2];
sum(...myNumbers);

```

## Short Circuiting (&& and II)
Go read about `Truthsy` and `Falsy` values in JS.
Whats the use of short Circuiting and where it is used? // imp to understand.
```Javascript
console.log(undefined || 0 || '' || false || null || 'hi'); // hi

// Short circuiting used to assign values on RunTime.
let a = 5;
let b = a ? a : 10; // if 'a' exists assign value of 'a' to 'b' otherwise assign 10 to 'b'
console.log(b); // 5

let c = a || 15; // if a is truthsy value then assign it to 'c' otherwise assign 15.
console.log(c); // 5

// && operator - 
console.log('Hello' && 23 && null && undefined && 'jonas'); // null

let a = 10;
let c = (a && 'hello');
console.log(c); // hello
```
